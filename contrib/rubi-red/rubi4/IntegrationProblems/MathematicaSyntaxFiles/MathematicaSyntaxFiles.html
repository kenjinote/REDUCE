Indefinite Integration Test Suite Problems

Expressed In Mathematica Syntax


This page provides links to Mathematica package files containing the indefinite integration suite problems expressed in Mathematica syntax. Each problem is stored as a list of the following items:
    1. the integrand;
    2. the integration variable;
    3. the number of rule applications required to integrate the integrand;
    4. the optimal antiderivative; and
    5. on a few problems, a nonoptimal, but acceptable, antiderivative.
Mathematica is required to load these files; however, they can also be viewed as text files. To view or download the test suite files, click on one of the following links:
